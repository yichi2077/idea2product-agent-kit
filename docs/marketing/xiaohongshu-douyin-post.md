**正文**

事情是这样的。

上周我把 Anthropic 那篇《创始人手册》(Founder's Playbook) 读完，读到一半停下来，盯着屏幕发了会儿呆。

它讲的东西特别朴素，先验证，再构建。就这六个字，放到 AI 编程上，突然变得有点刺耳。

手册里有句话像针一样扎我，42% 的创业公司，最后都死在同一件事上，做了个没人想要的东西。然后它补了一刀，AI 让这种死法更容易发生，更快了。

为什么。。。因为一个能跑的原型，太容易被你自己当成「已经验证了」。

愚钝如我，过去一年用 Claude Code、用 Cursor，一个想法上头，半个下午就能搓出个能跑的 demo。demo 亮起来那一刻真的很爽，我甚至觉得这玩意儿要火了。

结果呢，做完放那儿，没人用。

以前从想法到 demo，中间隔着设计、开发、调试、部署，你会被现实狠狠摩擦一遍。现在这层摩擦没了，AI 把执行成本一脚踩到地板。省下来的时间本该用来想清楚方向，可大多数时候，我只是更快地冲进了一个错的方向。

手册里还有一句，我截图存了下来，大意是，能跑的原型证明不了什么，真正算数的是你和用户之间那些对话。

我当时有点破防。我做的那些东西，动手之前从来没认真跟一个真人聊过。我以为我在验证需求，实际上我只是在用代码满足自己的冲动。

更阴险的是后面那句，确认偏误，现在配上了一台研究引擎。

你想啊，以前想自我欺骗，还得自己费劲找理由。现在你让 AI 帮你找支持的证据，它分分钟给你列十条，市场规模、竞品缺口、用户画像、趋势曲线，一整套看着像尽调报告的东西。你盯着那堆漂亮文字，越看越觉得自己有眼光。

很多时候我干的事，只是拿 AI 给自己的冲动做了一次精装修。

我合上文档，脑子里只剩一个念头。这套方法论是对的，可它是一篇文章。

文章这种东西，AI 会很客气地点头，然后转身继续帮你写代码。它不会拦你。

所以我花了几个晚上，做了个东西，叫 idea2product-agent-kit。

我想干的只有一件事，把手册里那些很对、又最容易被跳过的纪律，焊成一台 AI 绕不过去的状态机。

它把从想法到产品的过程切成 10 个阶段，从构思备忘、战略调研、战略决策、产品发现、产品定义，到验证原型、架构交付，再到特性规约、构建发布、效果复盘。中间卡了 4 道人工审批门。

你没把「为什么做、给谁做、凭什么做、怎么验证」用证据想清楚之前，它不会客气地陪你一路冲到写代码。它会拦你。

市面上的工具都在给 AI 拼命加油门。我给它配的是一脚刹车。

举个我自己最在意的设计。

跑到验证原型那一步，它会逼你做一件大部分人都想跳过的事，把核心交互做成最小的一个，拿去给大概 5 个真实用户用。别问他们「以后愿不愿意用」，这种问题只会换来一句客气话。你要盯着他们真实怎么操作，在哪儿停住，在哪儿皱眉，在哪儿真的愿意往下走。有信号就继续，信号弱就转向，场景特殊实在没法验证，就老老实实写清楚为什么豁免。

手册里那句「对话才是证据」，被我变成了一个你绕不开的步骤。

还有个我私心很喜欢的护栏，叫确认偏误护栏。你有一条关键假设，没跟任何用户聊过就关掉了，或者关掉它的时候只有支持的证据、一条反驳都没有，又或者你整个登记册攒下来的证据一边倒、支持的远多过反对的，它会在过门的时候，把这些当面给你标出来。

它不帮你做决定。它做的事情很简单，把你假装没看见的那条假设，重新摆回你眼前。

我觉得这事对 vibe coder 太重要了。现在很多人手太快了，一个想法刚冒头，IDE 已经打开。半天能跑，三天功能一堆，两周开始重构，一个月后发现根本没人要。最耗人的就是这段时间差，idea2product-agent-kit 想替你省下的，就是这几周。

它对普通用 AI 写东西的人，好处也挺实在。

全程本地优先，零额外 SaaS，状态和文档都躺在你自己的 Git 仓库里，想法不经过任何外部服务。

工具上也不挑，Claude Code、Cursor、Codex、OpenCode 这类对话式 agent 都能接，你不用换工作方式，对话框里说句 run p1 就开始了。

更值钱的是前面几个阶段攒下的那些产品定义、假设、竞品、架构约束，到真正写代码时会变成 AI 的事实依据，它有了白纸黑字可以对齐，幻觉和跑偏会少非常多。

它还顺手治了长项目一个老毛病，上下文崩溃。每次只让 AI 加载当前这一个阶段，不会把几万字历史一股脑塞进去，把模型脑子撑爆。

当然它不是万能的。紧急修个 bug 用不上它，高度耦合的老系统大手术也别指望它替你还清所有历史债。它适合从 0 到 1 的新项目、大型新功能，或者你准备认真 all in 的技术探路。

说实话，这个 kit 谈不上多聪明的设计，是 Anthropic 那篇手册把我点醒了，我只是把那份清醒写成了一段代码，让它每天替我守着那道闸。

它现在 MIT 开源，在我 GitHub 上，叫 idea2product-agent-kit。

如果你也是那种，想法一上头就忍不住先写代码的人，那我们大概是一类人。

下次再想冲之前，先给你的 AI agent 装上这脚刹车，让它问你一句。

这东西，真的有人要吗。

## 二、抖音口播版（约 55 秒）

市面上的工具都在给 AI 加油门，我给它配了一脚刹车。
