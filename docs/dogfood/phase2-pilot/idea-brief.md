# Idea Brief: idea2product Phase2 Feedback Loop Hardening

Build a closed-loop enhancement for idea2product so a solo operator can safely iterate upstream strategy and product artifacts without spec drift. The current product has strong phase gates, but downstream discoveries cannot formally reopen upstream phases, completed artifact changes are not detected, and PM critic / real evidence artifacts are advisory rather than state-machine preconditions.

The target user is the same solo operator described by the kit: one person acting as strategist, PM, and engineer. The desired outcome is a safer pipeline where rework is explicit, stale artifacts are visible, and human gates receive evidence that the relevant reviews actually happened.
