# Pipeline Map

- `.pipeline/recipes`: closed phase recipes with required, conditional, optional, external-tool, and non-skill process lists.
- `.pipeline/state`: pipeline state, assumptions, risks, and decision log.
- `.pipeline/vendor`: audited canonical source for third-party prompt skills.
- `.agents/skills`: Codex discovery layer for active project skills.
- `docs/00-idea`: real idea input and P1 output.
- `docs/10-strategy`: P2-P3 strategy artifacts.
- `docs/20-product`: P4-P6 product and validation-prototype artifacts.
- `docs/30-architecture`: P7-P8 architecture and feature artifacts.
- `docs/40-delivery`: P9-P10 release and outcome artifacts.
